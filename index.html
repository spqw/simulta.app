Features

Everything you need to
run AI at scale.

Window Per Agent

Every agent runs in its own native macOS window. Resize, tile, full-screen, drag between Spaces — just like any app. No cramming everything into one terminal.

Alt-Tab Switching

Press Option-Tab to cycle through agents exactly like macOS app switching. Your muscle memory works. Each window shows a live preview in the switcher.

Custom Viz Panels

Drop in any HTML/JS/CSS as a visualization. File trees, test dashboards, data flows, architecture diagrams — your agents' work rendered beautifully in real-time.

Multi-Provider

Claude Code, Codex, Gemini CLI, Aider, OpenCode — run any CLI agent. Mix providers in the same workflow. Route tasks to the best model for the job.

Workflow Orchestration

Define agent pipelines: Agent A writes code, Agent B writes tests, Agent C reviews. Dependencies resolve automatically. Fan-out for parallelism.

Smart Notifications

Native macOS notifications when agents need approval, finish tasks, or hit errors. Works with Focus modes. Approve tool requests from the notification banner.

Global Search

Search across all agents, all sessions, all messages in under 50ms. FTS5 + BM25 ranking. Find that one conversation where the agent fixed the auth bug.

Menu Bar Control

Always-on menu bar shows active agents, pending approvals, and error counts. One-click approve/deny. Batch operations. Never miss a blocked agent.

100% Local

Everything stays on your machine. No cloud, no telemetry, no accounts. Your code, your agents, your data. The server runs on localhost. Period.

FAQ

Questions & answers.

What agents does Simulta support?

Any CLI-based AI agent: Claude Code, Codex CLI, Gemini CLI, Aider, OpenCode, and any custom agent that runs in a terminal. If it has a CLI, Simulta can manage it.

How is this different from tmux or terminal tabs?

Simulta gives each agent a native macOS window with live visualization, status tracking, progress rings, global search across all sessions, Alt-Tab switching with previews, orchestration pipelines, and a menu bar for approvals. It's purpose-built for AI agents — not a generic terminal multiplexer.

How do the visualizations work?

Each agent window has a visualization panel that renders HTML/JS/CSS in real-time. Built-in viz types include file tree, test runner, data flow, and architecture graph. You can also create custom visualizations with the Viz SDK — drop in any HTML file and it receives a stream of agent events.

How many agents can I run simultaneously?

Tested up to 50+ concurrent agents on a MacBook Pro M3 Max with 36GB RAM. The actual limit depends on your hardware and what each agent is doing. Simulta's resource scheduler prevents overload.

Is my data sent to any server?

No. Simulta runs entirely on your machine. The search index is a local SQLite database. There is no cloud sync, no telemetry, and no accounts required. Your code and agent sessions never leave your computer.

Can agents communicate with each other?

Yes. Simulta's orchestration engine lets you define workflows where agents pass context to each other. Agent A finishes code, Agent B receives those files for testing, Agent C reviews the PR. You define the pipeline, Simulta handles the wiring.

What macOS version do I need?

macOS 14 (Sonoma) or later. Works on both Apple Silicon and Intel Macs. Optimized for Apple Silicon.

Is there a Windows or Linux version?

Not yet. Simulta is macOS-native (SwiftUI) to deliver the best possible experience. We're exploring cross-platform options but no timeline yet.
