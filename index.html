Global Search

Search across all sessions, messages, tool calls, tool outputs, and file diffs. FTS5 with BM25 ranking. Filter by project, provider, date, and message type. Results in under 50ms.

Native macOS

Menu Bar

Your data stays on your machine. The search index is local SQLite. No cloud sync, no telemetry, no accounts required. The server runs on localhost.

FAQ

Common questions.

What agents does Simulta support?

Claude Code, Codex CLI, Gemini, and OpenCode. We add new providers as they become available. All providers work with the same search index and unified UI.

Is my data sent to any server?

No. Simulta runs entirely on your machine. The search index is a local SQLite database. The server runs on localhost. There is no cloud sync, no telemetry, and no accounts required. Your session data never leaves your computer.

How does the search work?

Simulta uses SQLite FTS5 with BM25 ranking. It watches your session JSONL files and indexes every message, tool call, tool output, and file diff in real-time. Queries return in under 50ms even with tens of thousands of documents. You can filter by project, provider, message type, and date range.

Why native instead of a web app?

Native means instant launch, minimal memory usage (typically under 80MB), real macOS notifications that work with Focus modes, menu bar presence, keyboard shortcuts that don't conflict with the browser, and proper system integration. Electron apps can't match this.

What macOS version do I need?

macOS 14 (Sonoma) or later. Works on both Apple Silicon and Intel Macs.

Can I use it alongside Yep Anywhere?

Yes. Simulta connects to the same session files that Yep Anywhere reads. You can use both simultaneously — Simulta on your Mac, Yep Anywhere on your phone. They don't conflict.

What does "lifetime" mean?

Pay once, use forever. You get all current features and all future updates for the lifetime of the product. No recurring charges, no expiration.

Is there a refund policy?

Yes. If you're not satisfied within 30 days of purchase, we'll refund you in full. No questions asked.
